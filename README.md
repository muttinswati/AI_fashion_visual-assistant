# 👗 AI Fashion Visual Assistant

A smart visual search tool designed to help users find matching apparel from a massive collection of **44,000+ fashion items**. 
Simply upload an image, and the assistant identifies the closest visual matches in seconds.

## ✨ Key Features
* **Visual Discovery:** Upload any outfit image to find similar products instantly.
* **Instant Retrieval:** Optimized to search through 44k items with sub-second latency.
* **Smart Metadata:** Integrated product details including category, gender, and brand.
* **Clean Interface:** A modern React-based UI designed for a smooth user experience.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:**  (Python) & Fastapi
* **ML/AI:** Scikit-learn  TensorFlow/Keras 
* **Data:** Pandas, NumPy, CSV Metadata
